Smart Air (STAIR)

STAIR

This project plots ambient particulate matter, CO2, air pressure, temperature, and humidity data from sensors onto a map. Our current deployment of devices are built using the SAML11 microcontroller. We use AWS as a backend.

SIC2

SIC2

SIC2, or Securing Micocontroller based IoT Devices with Crypto Coprocessors, is a framework for secure key provisioning that protects end users' private keys from software attacks and untrustworthy manufacturers. Our case study is the ESP32 MCU along with the ATECC608A crypto processor. We also present novel software attacks against this MCU. Technical report and source code are below:
